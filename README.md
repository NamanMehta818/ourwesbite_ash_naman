# OpenSite Labs

A clean, modern landing page for a nonprofit that builds free websites for small businesses.

## Files

- `index.html` — Landing page with hero, services, process, and contact form.
- `styles.css` — Modern, techy styling with dark gradients and glassy panels.
- `script.js` — Mobile navigation toggle and demo form submission.

## Run locally

1. Open `index.html` directly in your browser.
2. Or run a simple server from the terminal:

```bash
python -m http.server 8000
```

Then open: http://localhost:8000

## Customize

- Update the copy in `index.html` to match your nonprofit name, mission, and application details.
- Replace the `alert()` in `script.js` with a real form handler (email/post request) when you're ready.
